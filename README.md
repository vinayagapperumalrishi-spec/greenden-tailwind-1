🌿 Greenden Website

Greenden is a modern and responsive plant-themed website built using Tailwind CSS. This project focuses on delivering a clean and visually appealing interface for plant lovers, showcasing greenery, products, and eco-friendly ideas.

The website consists of two main pages, designed with a consistent layout and smooth navigation. Each page is styled using Tailwind’s utility-first approach, ensuring responsiveness and a seamless user experience across different devices.

The project highlights practical frontend development skills such as responsive design, layout structuring, and efficient styling without writing custom CSS.

📄 Pages Included
Home Page – आकर्षक landing section with featured plants and introduction
Products Page – Displays plant collections with a clean grid layout

Fully responsive design
Modern and minimal UI
Built entirely with Tailwind CSS
Consistent design across multiple pages
Fast and lightweight
